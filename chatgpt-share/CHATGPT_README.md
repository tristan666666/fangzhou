# 给 ChatGPT 的说明包

这个文件夹是专门给外部 `ChatGPT` 看的，不依赖 `localhost`。

你可以直接把下面这两个文件上传给 ChatGPT：

- `PRODUCT_CONTEXT.md`
- `demo-export.html`

## 推荐你发给 ChatGPT 的一句话

请阅读我上传的产品说明和静态页面导出，帮我从产品结构、信息架构、交互流程和 UI 可用性四个角度，评估这个“红人BD任务系统”应该怎么继续优化。不要泛泛而谈，要具体指出页面里哪些区域不合理、为什么不合理、应该怎么改。

## 文件说明

### 1. `PRODUCT_CONTEXT.md`

这是产品说明文档，重点解释：

- 这个系统的业务本质
- 核心对象
- 页面结构
- 当前页面逻辑
- 你希望 ChatGPT 重点帮你看的问题

### 2. `demo-export.html`

这是一个独立静态页面，双击就能打开。

它不是完整运行版系统，但已经把当前 demo 的核心结构抽出来了：

- 创建任务页
- 任务执行页
- 会话处理页
- 任务草案区
